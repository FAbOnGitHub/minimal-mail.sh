# minimal-mail.sh


## Goal

This a minimal mailer in CLI for bash via php command because in some hosts
something seems wrong.

## Usage


```bash
$ME <from> <to> '<subject>' < body_via_stdin
```
```bash
date | minimal-mail.sh santa@christmas.org little@boy.org 'Your list' 
```
